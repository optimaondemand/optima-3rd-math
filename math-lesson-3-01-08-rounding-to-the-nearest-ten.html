📐 Key Concept — The Pattern Is Now Complete

Same rule, third (and finest) scale.

Ones digit 0–4 → round down.  ·  Ones digit 5–9 → round up.

The unit's pattern in one sentence: The deciding digit is always one place to the right of where you're rounding.

Nearest thousand → deciding digit is hundreds. Nearest hundred → deciding digit is tens. Nearest ten → deciding digit is ones.

Word problem

A hiking trail along a ridge is 4,374 meters long. Round its length three ways — and notice how each version tells a different story.

Nearest TEN:
Nearest HUNDRED:
Nearest THOUSAND:
✅ All three correct! Nearest ten: 4,370 (ones=4, round down). Nearest hundred: 4,400 (tens=7, round up). Nearest thousand: 4,000 (hundreds=3, round down). One trail, three "headline" lengths — each correct, each useful for a different audience.
C
Cora — checking
"Notice how nearest ten and nearest hundred move in OPPOSITE directions for the same number. 4,374 rounds DOWN to 4,370 but UP to 4,400. Each scale checks a different deciding digit."
🔄 Three rounds, three deciding digits.
Nearest TEN: ones digit decides → look at 4. Nearest HUNDRED: tens digit decides → look at 7. Nearest THOUSAND: hundreds digit decides → look at 3.

📐 Key Concept

Every bar model holds three pieces of information: two parts and a whole.

If you know both PARTS, ADD them to find the WHOLE. If you know the WHOLE and one PART, SUBTRACT to find the missing PART.

The picture itself tells you which operation to use. Draw the bar before you start the math.

Worked Example: A Different Kind of Bar
When the question compares two numbers.
Problem

Two classes raised money for a field trip. The sum of their amounts is $640. The greater class raised $397. How much did the other class raise — and what's the difference between them?

A
Atlas — drawing it
"The sum is $640. That's the whole. One part is $397 — the greater class. The other class is the part I don't know yet. I'll draw a bar split into the known part and the unknown part."
H
Hazel — finding the missing part
"The whole is 640 and one part is 397. To find the other part, I subtract: 640 − 397. Using round-and-adjust: 640 − 400 = 240, then add 3 back. Other class raised $243."
F
Felix — the difference
"Now I know both class amounts: $397 and $243. The DIFFERENCE between them is 397 − 243. Using break-apart: 397 − 200 = 197, then 197 − 43 = 154. The difference is $154."
C
Cora — checking the work
"Let's check both answers. 397 + 243 should equal 640. 397 + 243 → 400 + 243 − 3 = 643 − 3 = 640. ✓ And the greater minus the other: 397 − 243 = 154 ✓."

Other class: $243  ·  Difference: $154

💡

One bar — two answers. The picture holds both the "missing part" and the "difference" at the same time.

Today's Mission
Build a 3-digit number on the mat, multiply each column, trade when a column overflows.
❓
Essential Question
Does adding a hundreds column change anything?

No. Same rule: multiply each column. If a column overflows, trade up. The hundreds column just adds one more place to build in.

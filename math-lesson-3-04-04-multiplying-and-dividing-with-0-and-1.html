Essential Question
What do 0 and 1 do?

Multiply by 0 → always 0 (zero groups of anything is nothing). Multiply by 1 → unchanged. Divide by 1 → unchanged. Divide a number by itself → 1.

Two simple numbers, four predictable patterns.

📐 Key Concept

The four rules:

N × 0 = 0 · N × 1 = N · N ÷ 1 = N · N ÷ N = 1 (when N ≠ 0)

You can NEVER divide by 0 — zero groups can't share anything. That math has no answer.

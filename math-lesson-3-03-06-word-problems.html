📐 Key Concept

The multi-step recipe:

1. Read it twice.   2. Find the hidden step.   3. Pick a tool and solve step 1.   4. Use that answer in step 2.   5. Sense-check with an estimate.

Every two-step word problem has the same shape: one question hides inside another. Once you spot it, the rest is just doing the math you already know.

Worked Example: Two Stores, Two Steps
All Unit 3 tools combined.
Problem

Store A used 3,408 lights for a holiday display. Store B used 1,275 fewer lights than Store A. How many lights did the TWO STORES use ALTOGETHER?

A
Atlas — plan first
"I need the TOTAL of two stores. I know Store A (3,408). Store B is given by 'fewer than A' — that's the hidden step. So: Step 1, find Store B; Step 2, add A and B."
H
Hazel — solve step 1
"Step 1: Store B = 3,408 − 1,275. Stack and subtract. Ones: 8−5=3. Tens: 0−7 → unbundle. Hundreds 4→3, tens 0→10. 10−7=3. Hundreds: 3−2=1. Thousands: 3−1=2. Store B = 2,133."
Two steps, in order
1️⃣ Store B (hidden step): 3,408 − 1,275 = 2,133
2️⃣ Total (real question): 3,408 + 2,133 = 5,541
✓ Sense check: 3,400 + 2,100 ≈ 5,500. The exact answer 5,541 lands right where the estimate predicted.
F
Felix — solve step 2
"Step 2 uses my answer from step 1. Total = 3,408 + 2,133. Stack and add. Ones: 8+3=11, write 1, carry 1. Tens: 0+3+1=4. Hundreds: 4+1=5. Thousands: 3+2=5. Total = 5,541."
C
Cora — sense check
"Estimate-check: 3,400 + 2,100 = 5,500. My exact 5,541 is right next to 5,500. ✓ Reasonable. If I had gotten 7,541 or 2,541, the estimate would have caught it."

Both stores used 5,541 lights.

💡

Every step used a Unit 3 tool: column subtraction with unbundling (step 1), column addition with regrouping (step 2), estimation (sense check). You used the whole unit in one problem.
